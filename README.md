D&D Combat Tracker MVP
Een minimalistische, mobile-first webapp voor D&D-spelers (5e editie).
Features
• Play/Edit Mode: Wissel tussen het bijhouden van je sessie en het aanpassen van je karakterdata.
• Action Economy: Overzicht van Actions, Bonus Actions en Reactions.
• GitHub Sync: Sla je karakterdata en sessienotities direct op in deze repository via de GitHub API.
• Obsidian Ready: Notities worden opgeslagen in Markdown-formaat.
Setup
1.	Upload de index.html naar de root van je repository.
2.	Ga naar Settings > Pages in deze GitHub repo.
3.	Selecteer de main branch en klik op Save.
4.	Je app is na een minuutje live op de getoonde URL!
GitHub Integratie
Om de sync te laten werken:
1.	Maak een Personal Access Token (classic) aan met repo rechten.
2.	Open de app, ga naar Edit Mode (potloodje).
3.	Voer je token en deze repo naam (Mosa-1994/dnd-character-app) in.
